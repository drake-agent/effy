# Effy 사용 가이드

> 팀의 두뇌, AI가 구동합니다

---

## Effy가 뭐야?

Effy는 팀 전용 AI 어시스턴트입니다. 팀의 대화를 학습하고, 결정사항을 기억하며, 맥락을 이해합니다.

**개인 ChatGPT와 다른 점:**
- 팀 전체 대화 맥락을 기억합니다
- 누가 어떤 주제에 전문인지 파악합니다
- 이전에 결정된 사항을 추적합니다
- 팀원마다 다른 아침 브리핑을 보냅니다

---

## 기능 및 예시

### 1. 일반 질문/대화

아무 질문이나 편하게 하세요. 일반적인 AI 어시스턴트처럼 동작합니다.

```
"이번 분기 OKR 작성할 때 참고할 프레임워크 알려줘"
"파이썬으로 엑셀 파일 읽는 코드 짜줘"
"이메일 초안 써줘. 거래처에 미팅 일정 변경 요청"
```

### 2. 기억하기

중요한 내용을 Effy에게 기억시킬 수 있습니다. 기억된 내용은 팀 전체가 검색할 수 있습니다.

```
"오늘 회의에서 디자인 시스템 v2로 가기로 결정했어. 기억해"
"다음주 수요일까지 QA 완료해야 해. 기억해둬"
"API 응답 포맷을 camelCase로 통일하기로 했어. 저장해"
```

### 3. 기억 검색

이전에 저장한 내용이나 팀 대화에서 나온 결정사항을 찾아줍니다.

```
"지난번에 API 포맷 어떻게 하기로 했지?"
"디자인 시스템 관련 결정사항 알려줘"
"저번 주에 QA 일정 뭐라고 했었지?"
```

### 4. 전문가 찾기

팀 내에서 특정 주제에 대해 누가 잘 아는지 찾아줍니다.

```
"React 관련해서 누가 잘 알아?"
"데이터 분석 관련 전문가 누구야?"
"인프라 셋업 관련해서 누구한테 물어봐야 해?"
```

### 5. 아침 브리핑 (매일 오전 9시)

매일 아침 나에게 맞는 브리핑을 DM으로 보내줍니다.

- 내 부서 관련 결정사항
- 나를 멘션한 대화
- 내 전문분야 관련 미답변 질문
- 진행 중인 프로젝트 업데이트

> 100명이 쓰면 100명 각각 다른 브리핑을 받습니다.

### 6. 자동 요약

채널에서 스레드가 길어지면 (10개 메시지 이상) 자동으로 핵심만 요약합니다.

### 7. 중복 질문 감지

이전에 답변된 유사한 질문이 있으면 자동으로 찾아서 연결해줍니다.

## 사용 방법

| 상황 | 방법 |
|------|------|
| **개인 채팅** | Effy 앱을 열고 바로 메시지 |
| **채널에서** | `@Effy` 멘션 후 질문 |
| **기능 안내** | `help` 입력 |
| **프로필 수정** | `내 프로필 수정` 입력 |

---

## Effy의 대화 수집 범위

Effy는 **모든 대화를 무차별 수집하지 않습니다.**

| 상황 | 수집 여부 | 설명 |
|------|---------|------|
| `@Effy` 멘션한 채널 대화 | ✅ 수집 | Effy에게 직접 질문한 내용 |
| Effy와 1:1 DM | ✅ 수집 | 개인 대화 내용 |
| Effy가 설치된 채널의 일반 대화 | ⚠️ 선별 수집 | Observer가 중요 결정사항만 감지 |
| Effy가 없는 채널 대화 | ❌ 접근 불가 | 볼 수 없음 |
| 팀원끼리 1:1 DM | ❌ 접근 불가 | 볼 수 없음 |

### 데이터 저장 위치

- 모든 데이터는 **회사 AWS 내부 PostgreSQL**에 저장됩니다
- 외부 클라우드에 데이터가 나가지 않습니다
- AI 응답 생성 시에만 Anthropic API로 메시지 내용이 전송됩니다 (저장 안 함)

## 자주 묻는 질문

**Q: Effy가 내 모든 대화를 보고 있어?**
A: 아닙니다. Effy가 설치된 채널과 Effy에게 직접 보낸 메시지만 접근 가능합니다.

**Q: 내가 기억시킨 내용을 다른 팀원도 볼 수 있어?**
A: 네, 팀 공유 메모리에 저장되어 다른 팀원이 검색할 수 있습니다. 개인적인 내용은 기억시키지 마세요.

**Q: 아침 브리핑은 어떻게 받아?**
A: Effy와 한 번이라도 대화하면, 매일 오전 9시에 자동으로 DM이 옵니다.

**Q: Effy가 틀린 답변을 하면?**
A: "아니야, 그게 아니라..." 라고 교정해주세요. Effy는 교정 내용을 학습해서 같은 실수를 반복하지 않습니다.
